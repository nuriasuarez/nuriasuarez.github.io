The structure of an HTML document*

<!DOCTYPE html>
"Document Type Declaration" - It is the first thing in your HTML document. t is an instruction to the web browser about what version of HTML the page is written in.
<html lang="en"> </html>
This is a wrapper around the entire document. The lang attribute specifies the primary language for the page.
<head> </head>
This is a wrapper that contains other information about the document such as internal or external styles and scripts.
<body> </body>
Contains the visible part of the page.
