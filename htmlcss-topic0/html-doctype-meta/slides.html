Doctypes & Metatags

DOCTYPE

HTML 4.01 Transitional DTD: 

        <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" 
"http://www.w3.org/TR/1999/REC-html401-19991224/loose.dtd">

HTML5: 

      <!DOCTYPE html>

DOCTYPE - Where should it be?

The doctype isn't an actual tag, but it needs to be at start at every HTML page to tell browser which version of HTML you're using.

HTML Validators

Browsers will often render HTML that has mistakes in it, using different error correction strategies.

To check that your HTML is actually valid according to the standard, run it trough the W3C validator

You can also install a validator add-on for Firefox

Meta tags

<head>
<meta charset="UTF-8">
<meta name="description" content="Free Web tutorials">
<meta name="keywords" content="HTML,CSS,XML,JavaScript">
<meta name="author" content="Hege Refsnes">
</head>

Metadata is data (information) about data.

Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.

Meta viewport tag

Syntax

<meta name="viewport" content="name=value,name=value">

Example

<meta name="viewport" content="width=device-width,initial-scale=1">
