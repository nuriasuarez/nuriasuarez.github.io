CSS: Exercise 1

  1. Create an empty styles.css file and link it to the index.html using tag
  2. Include normalize.css before your style.css to fix cross-browser problems.
  3. First figure out what selector you can use to select a part of the page, adding class or id attributes to the HTML if necessary.
  4. Starting with the solution from the HTML exercise 3, add CSS rules to the webpage so that it looks like the screenshot below.
  5. Then create the CSS rule with that selector and the color property (e.g. color: red). Useful color names are red, blue, green, yellow, purple, orange, darkred, and lightblue.
  6. Add a global font definition (at html element) with a value of 14px, using a font-family you like.
  7. Note that there are often multiple ways to achieve the same result in CSS. Try to use a mix of the different types of selectors- id, class, position in document, etc.
