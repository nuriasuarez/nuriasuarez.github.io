CSS: "Cascading Style Sheets"

  • A language that lets you style the elements on your page
  • Consists of "style rules"
  • Each style rule consists of a "selector" and "declarations" of property-value pairs:
selector {
 property: value;  
 property: value;
}

Universal selector

* {
   margin: 0;
   font-size: 20px;
   line-height: 25px;
}
